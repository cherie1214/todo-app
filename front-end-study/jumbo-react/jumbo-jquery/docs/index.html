How to Use It?

In order to do development with it, first you'll need to have the following tools installed:

These tools are for development purpose only, you don't need them in production environment, meaning you can use any back-end languages you like, NodeJS is just one of the options, not required

  • Node.js >=v6.5.0 - For npm as a package manager (NPM comes along with it), just download and install it, that's it.
  • Ruby - For Writing Sass (if you are using a Mac, you already have it), Sass for Scss CSS.

A few steps to get started:

1. Enter into "source" folder with your command line tool.

$ cd /path-to-source-folder

2. Install all the NPM packages with command: 

$ npm install

3. Then you can start browsing the HTML pages. index.html is the right place to start with.
